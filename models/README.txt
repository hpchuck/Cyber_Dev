For a professional 3D character model, please download a GLTF/GLB file from one of these sources:

1. Sketchfab (free models available): https://sketchfab.com/3d-models?features=downloadable&sort_by=-likeCount
2. Mixamo (animated characters): https://www.mixamo.com/
3. TurboSquid (free & paid models): https://www.turbosquid.com/Search/3D-Models/free/character
4. CGTrader (free & paid models): https://www.cgtrader.com/free-3d-models/character

After downloading, place it in this directory (/public/models/) with the name 'character.glb'.

Recommended model specifications:
- Format: glTF 2.0 (.glb)
- Polycount: Under 50k triangles for good performance
- Rigged: If you want animations
- PBR Materials: For best visual quality
- Size: Under 5MB for fast loading

A small placeholder model is currently in place to ensure the website functions correctly while you select a permanent character model.
